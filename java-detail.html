    我们80%的同学都接近零基础，另外20%同学自以为的基础也可忽略不计。同学中50%的同学甚至之前从未接触过代码比如跨专业、转行，所以从统计角度来看你应该也不会是例外 当然有过编程经历会让你少走弯路，前进的更顺畅，对于完全的零基础要做好受虐的准备 能不能学好关键是看性格，如果你能踏实的做好一件事，就能学好前端顺利毕业

  • 毕业的同学待遇如何？

    同学的普遍待遇为6k 到 15k 之间，只要勤奋主动，待遇超过10k很正常。待遇的差异主要源自工作城市差异、个人学习态度差异、个人技术背景差异 那些期望毕业后立即拿到20k、40k的同学暂时会让你失望，除非你有好的背景和机缘(先前的工作经历、个人本身很优秀、在美帝等)
